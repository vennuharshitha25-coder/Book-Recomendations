# 📚 Book Recommendation System

A simple and interactive web application that recommends books based on user-selected genres.
Built using **Spring Boot** for backend and **HTML + CSS** for frontend.

---

## 🚀 Tech Stack

* ☕ Java
* 🌱 Spring Boot
* 🌐 HTML
* 🎨 CSS

---

## ✨ Features

* 📖 Genre-based book recommendations
* 🎭 Supports multiple genres (Romance, Thriller, Comedy, Cultural, etc.)
* 📝 Displays book title with a short storyline
* 🔍 Case-insensitive search
* 💻 Simple and user-friendly interface

## ▶️ How to Run the Project (VS Code)

1. Clone the repository

   ```
   git clone https://github.com/your-username/Bookrecommendationsapp.git
   ```

2. Open the project in **VS Code**

   * Open VS Code
   * Click **File → Open Folder**
   * Select your project folder

3. Install required extensions:

   * Extension Pack for Java

4. Run the application:

   * Open `DemoApplication.java`
   * Click **Run ▶️** (top right)
     OR
   * Right click → **Run Java**

5. Open browser and go to:
   👉 http://localhost:8081

---

## 💻 Alternative Run (Terminal)

You can also run using terminal in VS Code:

```
./mvnw spring-boot:run
```
